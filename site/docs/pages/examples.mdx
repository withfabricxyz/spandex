# Examples

A growing list of projects built with SpanDEX. If you have built something cool, please
consider submitting a PR to add it here!

## Web and Mobile Apps
* [spanDEX Swap Demo](https://sip-app-production.up.railway.app/swap) - Swap interface which compares performance of multiple aggregators and continuously provides the best price
* [SLAP Bench](https://quote-bench-production.up.railway.app/) - A benchmarking tool to compare SLAP (Stability, latency, accuracy, price) performance across aggregators and chains over time
