## Configuration

<Callout type="info">
  0x requires an API key for access. You can obtain one by [signing up](https://dashboard.0x.org/create-account).
</Callout>

<Callout type="info">
  If you have negotiated surplus sharing with 0x, set the `enableSurplus` option to true.
</Callout>

```ts twoslash
import { createConfig } from "@withfabric/spandex";

export const config = createConfig({
  providers: {
    '0x': {
      // required!
      apiKey: "YOUR_0X_API_KEY",
      // optional, defaults to false
      enableSurplus: false,
    },
    // ...
  },
});
```
