# Security Policy

## Reporting a Vulnerability

If you believe you have found a security issue, please report it privately.

### Urgent Issues

Email us at `oss@withfabric.xyz` and include:
- A clear description of the issue.
- The assessed severity (e.g., low/medium/high/critical).
- A minimum reproducible example (steps, code, or request details).

We will acknowledge receipt and work with you on a coordinated fix.
